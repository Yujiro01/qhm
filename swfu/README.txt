■SWFUploadとは？
----------------------

ファイルアップロードは、１ファイルずつ選ぶ必要があり、
とても面倒です。
そこで、Flash(swf)を使って、複数ファイルを選択できるようにし、
AJAX(だと思う)で、ファイルを順次転送するプログラムが、
SWFUploadです。

開発者情報は、こちら
http://www.swfupload.org/


■SWFUpload Frontとは？
----------------------

SWFUploadは、とても便利ですが、単に使いたい人には敷居が高い
(かなり高い？)ので、シンプルで高速なファイル管理ソフトとして、
操作部分(Front)を付けたものです。

基本的に、Quick Homepage Makerで画像表示や、
ファイルダウンロードのために、画像、PDFなどのアップロードを
便利にするために開発(というほど、大げさではない)しました。

Quick Homepage Maker
http://www.open-qhm.net/

■インストール方法
----------------------

1. ダウンロードしたファイルを解凍する
2. フォルダ名を決める(変更しなければ、swfu) [BASE]
3. 任意の場所にアップロード
4. システム内の"d"フォルダをPHPで書き込み可能権限にする(777)


■設定
----------------------

config.phpを編集します。
若干のプログラムファイルを読める能力が必要です。

※ユーザー名、パスワードは編集して下さい
※認証なしは、やめましょう。昨今のSPAMなどの被害がありますので

USER : 英数半角でユーザー名を入れる
PASSWD : md5でハッシュ化したパスワードを入れる
ここで、求めるといいかも
http://www.open-qhm.net/?cmd=md5


BASE : 設置先フォルダを編集する
DATA_DIR : 保存先をd以外に変更した場合
ROW, COLS : 表示数の制御
ENABLE_QHM_MODE : QHMのためのコマンドを表示(1)、非表示(0)


■設定例　：　友達と写真を共有
--------------------------

1. swfupload frontを設置
2. フォルダ名を好きな物に変更(shareとか？)
3. config.phpのBASEをshareに変更
4. config.phpのENABLE_QHM_MODEを0に変更
